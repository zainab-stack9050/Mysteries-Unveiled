# Mysteries-Unveiled

Frontend website showcasing unsolved mysteries with dynamic JSON-driven content pages.

##  Live Preview
> Open `index.html` in your browser — no server required.

##  Features
- Multi-page navigation (Home, About, Contact)
- 10 unsolved mysteries across 5 categories
- Dynamic detail pages using URL parameters
- Mystery data loaded from external JSON file
- Responsive design with custom CSS styling

##  Categories
-  Ancient Mysteries
-  Science Mysteries
-  Space Mysteries
-  Missing People
-  Unexplained Phenomena

##  Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)
- JSON

##  Project Structure
Mysteries-Unveiled/
├── index.html        # Homepage with all mystery cards
├── mystery.html      # Dynamic detail page (loads from URL params)
├── about.html        # About the project & authors
├── contact.html      # Contact page
├── mysteries.json    # All 10 mystery entries (data source)
├── mystery.js        # Fetches JSON & renders mystery details
└── style.css         # All styling & responsive design
