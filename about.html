About This Website

This website was created as part of a Web Development assignment. It showcases a collection of the world's greatest unsolved mysteries, each with a dedicated details page loaded dynamically using JSON and JavaScript.

The purpose of this project is to demonstrate:

  • HTML structure and multi-page navigation
  • CSS styling and responsive design
  • JavaScript DOM manipulation
  • Loading external JSON data
  • Using URL parameters to display dynamic content
